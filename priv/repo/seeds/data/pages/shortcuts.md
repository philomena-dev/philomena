Various functions of Philomena have keyboard shortcuts to make browsing easier and faster.

## Image Lists

| Key | Action                                                         |
| --- | -------------------------------------------------------------- |
| J   | Previous page                                                  |
| K   | Next page                                                      |
| F   | Favourite image (image under the mouse pointer)                |
| U   | Upvote image (image under the mouse pointer)                   |
| O   | View full image in current tab (image under the mouse pointer) |
| V   | View full image in new tab (image under the mouse pointer)     |

## Image Pages

| Key | Action                                                                                   |
| --- | ---------------------------------------------------------------------------------------- |
| J   | Previous image (in global index terms, the image posted after the one you're looking at) |
| K   | Next image (image posted before current one)                                             |
| R   | Random image                                                                             |
| S   | Open source URL                                                                          |
| L   | Open tags for editing                                                                    |
| I   | Go to index page containing the image being displayed                                    |
| F   | Favourite image                                                                          |
| U   | Upvote image                                                                             |
| O   | View full image in current tab (image under the mouse pointer)                           |
| V   | View full image in new tab (image under the mouse pointer)                               |
